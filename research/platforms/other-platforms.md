## GitHub Copilot

### Overview
- **URL**: https://github.com/features/copilot
- **Price**: $10/month (individual), $19/month (business)
- **Position**: Industry standard, most popular AI coding assistant

### Key Features
- Inline code suggestions
- Copilot Chat for explanations
- GitHub integration
- Works in VS Code, JetBrains, Vim, Neovim

### Strengths
- Best inline suggestions (feels natural)
- Learns your coding style
- Free for students and open source maintainers
- Large user base, proven reliability

### Weaknesses
- $10/month cost
- Privacy concerns (code sent to GitHub)
- Can suggest outdated patterns
- Less advanced agent features compared to Cursor/Windsurf

### Workshop Relevance: 8/10
- Good baseline for comparison
- Most participants likely familiar
- Free for students

## JetBrains AI

### Overview
- **URL**: https://www.jetbrains.com/ai/
- **Price**: Free tier (3 AI credits/30 days), from $8.33/month
- **Position**: Native AI integration for JetBrains IDEs

### Key Features
- Context-aware code generation
- Advanced code completion (unlimited on free tier)
- Automated test creation
- AI chat functionality
- Junie - new coding agent
- Local AI support

### Strengths
- Deep JetBrains IDE integration
- Unlimited code completion on free tier
- Local AI option
- Enterprise features

### Weaknesses
- Limited to JetBrains ecosystem
- Lower credit allowance on free tier
- Less advanced agent features

### Workshop Relevance: 7/10
- Excellent for JetBrains users
- Good local AI demonstration

## AWS CodeWhisperer

### Overview
- **URL**: https://aws.amazon.com/codewhisperer/
- **Price**: Free for individuals, $19/month (Pro)
- **Position**: AWS-focused AI coding assistant

### Key Features
- Excellent for AWS services
- Security scanning built-in
- Reference tracking (shows source)
- Free for individual developers

### Strengths
- Free for individuals
- Deep AWS integration
- Security scanning
- Code attribution

### Weaknesses
- AWS-focused (less general)
- Not as good as Copilot for general code
- Limited IDE support

### Workshop Relevance: 5/10
- Good for AWS-specific workshops
- Less relevant for general development

## Replit Ghostwriter / Replit AI

### Overview
- **URL**: https://replit.com/
- **Price**: Included with Replit subscription
- **Position**: Integrated AI in Replit's browser IDE

### Key Features
- Complete Code (autocomplete)
- Generate Code from descriptions
- Transform Code (refactor)
- Explain Code
- Ghostwriter Chat
- Replit Agent for autonomous building

### Strengths
- Integrated IDE experience
- Browser and mobile access
- Streamlined setup
- Can build complete applications

### Weaknesses
- Limited to Replit ecosystem
- Less powerful than dedicated tools
- Browser-based limitations

### Workshop Relevance: 6/10
- Good for beginners
- Excellent for web development workshops
- Easy setup (no installation)

## Workshop Selection Guide

### For General Development Workshops
**Recommended**: Cursor, Windsurf, or Continue.dev
- Most comprehensive AI features
- Best for demonstrating AI-native development

### For Enterprise/Privacy-Focused Workshops
**Recommended**: Tabnine, Cody
- Strong security and compliance features
- Enterprise-scale capabilities

### For Open-Source/Customization Workshops
**Recommended**: Continue.dev
- Fully open source
- Highly extensible
- Can be self-hosted

### For Beginner/Student Workshops
**Recommended**: GitHub Copilot (free for students), Replit AI
- Free or low-cost options
- Easy to get started
- Good learning resources
